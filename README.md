# AutoTouch
android 模拟屏幕点击。利用无障碍服务，并申请悬浮窗权限，全屏点击记录触控点，并定时重复模拟点击。
fork来之后增加了添加触摸点之后按照顺序执行的功能,适合傻瓜式的一些点击流程
